配置

有2种方法

第一种

在主题中添加参数

第二种

在 hexo 的 _config.yml 中添加参数

第二种配置会覆盖第一种

配置项

  • model 模型名称 默认值: z16
    • Gantzert_Felixander

  • width 宽度 默认值: 150
  • height 高度 默认值： 300
  • className <canvas> 元素的类名 默认值： live2d
  • id <canvas> 元素的id 默认值： live2dcanvas
  • bottom <canvas> 元素的底部偏移 默认值： -20
    如果嫌模型位置不正确 可以调整这个参数

新买的cc域名

发表于 2017-08-01 | 分类于 杂记

qo-op.cc
昨晚转了半天，本想买一个io后缀的域名，199每年实在是有点贵~最后转到阿里，发现cc域名又降价了？16一年的价格，就想买一个玩玩。
之前有一个ziyoo.cc现在看来好像很一般。选域名真的是一件很纠结的事情，太短的都被注册完了，
那我加连字符总可以了吧。。。###
从rui》mrui》rui-rui》ru-ru》画风突变~就变成o-o》u-u》i-i》巴拉巴拉，，，竟然竟然都被注册了，你们这群二次元死肥宅就只会卖萌，我吃柠檬，柠檬吃多了！！！
最后选的 qo-op.cc
